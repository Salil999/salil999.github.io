MangOS - Operating System

An implementation of an operating system that's written almost entirely from scratch. Areas that we focused on were file systems, hardware initializations, execute/halting programs, handling interrupts, scheduling, and multiple terminals.

Enrich - Hackathon Project

This is a hackathon project that my friends and I created at MHacks. The idea was to help people create their own personal investment portfolio and to invest passively. A small portion of your payment would be added on as extra and it would go towards that company's stock.
DevPost

eParking - Project Lead

This concept app was meant to simulate parking meters and authenticating purchases via credit card or mobile payment. The idea was to get rid of the use of coins and also be able to pay remotely. I worked mostly with the Raspberry Pi and used Python to communicate with a server that had a database.
Website

Home Control - Project Lead

Created a pebble smartwatch app that will get readings in your house using AllState's A6 IoT (Internet of Things) API. The app receives information from sensors that are placed in the house to detect potential problems in the home. DevPost

UIUCUMTD - Independent Project

Pebble smartwatch app that gets the current bus stops based on your current location and it will get the times for each bus stop as well. I mostly made this because I wanted to look at my watch to get the information instead of my cellphone. It's easier and faster.
Download App

Danger Detector

This was an iPhone application created at State Farm's HackDay 2015. Our team downloaded historical crime data in Chicago and used ForcaseIO's API to give the user a "danger level" based on their current location. The danger level would be detected from a radial distance around your current location.

Autonomous Vehicle

Created an autonomous vehicle using the Arduino Uno controller. The car was automated because we had put photo resistors on there so the car would follow a flashlight that was in front of it.
